# Portfolio

Dieses Projekt ist ein persönliches Portfolio von Raphael Neuberger. Es präsentiert Projekte, Fähigkeiten, Testimonials und Kontaktmöglichkeiten in einem modernen, responsiven Webdesign.

## Features
- Übersichtliche Startseite mit Hero-Sektion
- Mehrsprachigkeit (Deutsch/Englisch) über translations.js
- Responsive Design für alle Endgeräte
- Projektübersicht mit Bildern und Links
- Kontaktformular mit Datenschutz-Hinweis
- Testimonials von Kollegen
- Animierte Header-Navigation beim Scrollen

## Verzeichnisstruktur
- `index.html` – Hauptseite
- `header.html`, `footer.html` – Einbindung von Header und Footer
- `legalnotice.html`, `privacypolicy.html` – Rechtliche Hinweise
- `assets/img/` – Bilder für Projekte und Profil
- `css/` – Stylesheets für verschiedene Bereiche
- `js/` – JavaScript für Interaktivität und Übersetzungen

## Installation & Nutzung
1. Repository herunterladen oder klonen
2. Öffne die `index.html` im Browser
3. Für eigene Anpassungen: Passe die HTML-, CSS- oder JS-Dateien nach Bedarf an

## Technologien
- HTML5
- CSS3 (inkl. Flexbox, Grid, Media Queries)
- JavaScript (Vanilla)

## Hinweise
- Das Kontaktformular ist statisch und benötigt ggf. Backend-Anbindung für den Versand
- Die Übersetzungen können in `js/translations.js` angepasst werden
